# 🪐 The Relic Ring Protocol - Zeta-26 Network Rerouting

An efficient, resilient routing protocol simulation designed to reconnect the fractured Zeta-26 star system using legacy physical infrastructure. Built for the Launch26 Hackathon by IEEE Computer Society Chapter, University of Kelaniya.

## 🚀 Features & Milestones Achieved

* **M1: Universe Initialization:** Dynamically ingests `universe-config.json` to spin up the network architecture without any hardcoded planetary values.
* **M2: Multi-Hop Proof:** Traces packets across the cosmos, successfully handling Codex Translations (e.g., Base 5, Base 14) and converting data streams to ASCII for internal processing.
* **M3: Latency Breakdown:** Implements complex physical latency math including Subsurface Fiber Transit, Processing Tower Delay, Atmospheric Refraction, and Void Transmission.
* **M4: Chaos Test (Dynamic Rerouting):** Includes a Chaos Engine to simulate node failures. The protocol instantly detects dead zones and recalculates the lowest-latency path using Dijkstra's Algorithm without crashing.
